# Sales Invoice System / نظام فاتورة المبيعات

This repository contains the Agile planning, design, and testing documentation for a **Sales Invoice** module.

## 📁 Repository Files

| File | Description |
|------|-------------|

## 📌 Scope Overview

The documentation covers the full lifecycle of a sales invoice, including:

- **Master Data Setup** — branch selection, invoice numbering, payment methods, cash boxes, customers, and stores.
- **Item Details Grid** — item/barcode entry, F9 lookup, units, quantity validation, pricing, discounts, and tax.
- **Promotions & Totals** — item-level free quantities, invoice-level discounts, and dynamic totals.
- **Permissions & Search** — CRUD authorization and invoice search screen.
- **Global Settings** — system variables that control invoice behavior.

## 📌 نطاق المشروع

الوثائق تغطي دورة حياة فاتورة المبيعات بالكامل:

- **إعدادات الفاتورة والبيئة** — اختيار الفرع، الترقيم، طرق الدفع، الصناديق، العملاء، والمخازن.
- **إدخال تفاصيل الأصناف** — إدخال الصنف/الباركود، البحث F9، الوحدات، التحقق من الكمية، التسعير، الخصم، والضريبة.
- **العروض الترويجية والإجماليات** — الكميات المجانية على مستوى الصنف والخصومات على مستوى الفاتورة.
- **الصلاحيات والبحث** — صلاحيات CRUD وشاشة البحث عن الفواتير.
- **المتغيرات العامة** — الإعدادات العامة التي تتحكم بسلوك الفاتورة.
